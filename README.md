# Foot Manage

Application JavaFX de gestion d'un complexe de football : terrains, réservations, matériel, utilisateurs, historique et tournois.

## Documentation BTS SIO E6

- [Dossier des documents](./docs/README.md)
- [Dossier de correction](./correction/README.md)

## Structure utile

- [src/main/java](./src/main/java/) : code source Java
- [src/main/resources](./src/main/resources/) : vues FXML, styles et schéma SQL
- [docs](./docs/) : documents liés à l'E6
- [correction](./correction/) : éléments demandés pour la correction

## Lancement du projet

```bash
./mvnw javafx:run
```
